# 🎂 Online Cake Shopping Website

A web-based application that allows users to browse, customize, and order cakes online, providing a convenient and efficient shopping experience.

---

## 🚀 Overview

This project is an online cake ordering system where customers can browse different cakes, add them to a cart, customize orders, and place orders online. The system also includes an admin module to manage users and orders.

The goal is to simplify the cake ordering process and reduce manual effort.

---

## ⚠️ Problem Statement

Many bakeries maintain records manually, which leads to:

- Inefficient data management  
- Increased chances of errors  
- Difficulty handling large number of customers  

This system provides a digital solution for managing cake orders efficiently.

---

## 🎯 Objectives

- Develop an online platform for cake ordering  
- Allow users to browse and customize cakes  
- Provide admin functionality to manage data  
- Reduce manual work and improve efficiency  

---

## 🛠️ Technologies Used

- PHP  
- HTML  
- CSS  
- XAMPP (for server)  
- Notepad++  

---

## 📊 Key Features

- User registration and login  
- Browse available cakes  
- Add to cart and place order  
- Customize cake (flavour, size, etc.)  
- Admin panel to manage users and orders  
- Feedback system
